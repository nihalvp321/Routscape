# 📖 Overview

RouteScape is a comprehensive KSRTC Bus Booking and Tour Management System designed to simplify travel planning by integrating bus ticket booking, room reservations, and tour package management into a single platform.

The system enables passengers to search routes, book bus tickets, reserve accommodation, and explore tour packages efficiently. It provides a seamless travel experience while helping administrators manage transportation and tourism services effectively.

RouteScape serves as a digital solution for modernizing public transportation and tourism management through a user-friendly and centralized platform.

# 🚀 Installation Guide

## Clone Repository

```bash
git clone https://github.com/nihalvp321/routescape.git
cd routescape
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

## Open Browser

```text
http://127.0.0.1:5000
```
